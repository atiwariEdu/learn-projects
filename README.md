# Intern Self-Study Resources – Backend Track (Python + FastAPI)

Hey! While you're on break, here’s a curated list of the **best free video resources** to go from zero to building real FastAPI projects.  
Watch them in order, code along, and take notes. Everything here is beginner-friendly and up-to-date (2024–2025).

### 1. Python Basics & Mindset
If you’re completely new or need a solid refresh:
- [Python Full Course for Beginners (2025) – 6+ hours, excellent pacing](https://www.youtube.com/watch?v=iWS9ogMPOI0&t=124s)
